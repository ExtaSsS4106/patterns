📖 Обзор API

API для управления паттернами проектирования и их категориями. Все данные хранятся в Google Sheets и доступны через HTTP запросы.

🔗 Базовый URL

https://script.google.com/macros/s/ВАШ_ID_СКРИПТА/exec

Замените ВАШ_ID_СКРИПТА на ID вашего веб-приложения Google Apps Script.

📋 Формат ответа

Все ответы возвращаются в формате JSON со следующей структурой:

{
  "success": true/false,
  "data": {...},
  "error": "..."
}

🗄️ Структура Google Sheets

Для работы API необходимо создать два листа в Google Sheets:

  • Patterns - лист с паттернами проектирования
  • Categories - лист с категориями паттернов

🚀 Доступные эндпоинты

GET ?action=getAllPatterns

Получение всех паттернов с информацией о категориях.

Пример ответа: 
{
  "success": true,
  "data": [...] 
}
GET ?action=getPatternById&id={id}

Получение паттерна по ID.

id number
ID паттерна (обязательный)
GET ?action=getCategories

Получение всех категорий.

GET ?action=getPatternsByCategories&category_id={id}

Получение паттернов по категории.

category_id number
ID категории (обязательный)

📊 Структура: Patterns

Поле Тип Описание
idnumberУникальный идентификатор
titlestringНазвание паттерна (англ.)
category_idnumberID категории
termstringРусскоязычный термин
problemstringОписание проблемы
solutionstringОписание решения
examplesstringПримеры использования
consclusionsstringВыводы
