📖 Обзор API

API для управления паттернами проектирования и их категориями. Все данные хранятся в Google Sheets и доступны через HTTP запросы.

🔗 Базовый URL

https://script.google.com/macros/s/ВАШ_ID_СКРИПТА/exec

Замените ВАШ_ID_СКРИПТА на ID вашего веб-приложения Google Apps Script.

📋 Формат ответа

Все ответы возвращаются в формате JSON со следующей структурой:

{
  "success": true/false,
  "data": {...}, // данные при успешном запросе
  "error": "..." // сообщение об ошибке при failure
}

🗄️ Структура Google Sheets

Для работы API необходимо создать два листа в Google Sheets:

  • Patterns - лист с паттернами проектирования
  • Categories - лист с категориями паттернов

🚀 Доступные эндпоинты

GET ?action=getAllPatterns

Получение всех паттернов с информацией о категориях. Автоматически добавляет поле category_name из связанной категории.

Параметры:
action string (обязательный)
Должен быть равен "getAllPatterns"
Пример запроса: 
GET /exec?action=getAllPatterns
Пример ответа: 
{
  "success": true,
  "data": [
    {
      "id": 1,
      "title": "Singleton",
      "category_id": 1,
      "category_name": "Порождающие",
      "term": "Одиночка",
      "problem": "Необходимость гарантировать единственный экземпляр класса",
      "solution": "Создать класс с приватным конструктором и статическим методом",
      "examples": "Логирование, подключение к БД",
      "consclusions": "Простой способ контроля создания объектов"
    }
  ]
}

📊 Структура данных: Patterns

Таблица Patterns содержит полную информацию о паттернах проектирования.

Поле Тип Описание Обязательное
id number Уникальный идентификатор паттерна ✅ Да
title string Название паттерна (англ.) ✅ Да
category_id number ID категории (связь с Categories) ✅ Да
term string Русскоязычный термин ❌ Нет
problem string Описание проблемы, которую решает паттерн ❌ Нет
solution string Описание решения ❌ Нет
examples string Примеры использования ❌ Нет
consclusions string Выводы и рекомендации ❌ Нет
Пример объекта Pattern: 
{
  "id": 1,
  "title": "Singleton",
  "category_id": 1,
  "term": "Одиночка",
  "problem": "Необходимость гарантировать единственный экземпляр класса и глобальную точку доступа к нему",
  "solution": "Создать класс с приватным конструктором и статическим методом, возвращающим единственный экземпляр",
  "examples": "Логирование, пул соединений с БД, кэш",
  "consclusions": "Удобный паттерн для управления ресурсами, но требует осторожности в многопоточных средах"
}

📁 Структура данных: Categories

Таблица Categories содержит категории паттернов проектирования.

Поле Тип Описание Обязательное
id number Уникальный идентификатор категории ✅ Да
title string Название категории ✅ Да
Пример объекта Category: 
{
  "id": 1,
  "title": "Порождающие"
}

📌 Стандартные категории:

  • Порождающие (Creational) - паттерны создания объектов (Singleton, Factory, Builder, Prototype, Abstract Factory)
  • Структурные (Structural) - паттерны организации классов и объектов (Adapter, Bridge, Composite, Decorator, Facade, Flyweight, Proxy)
  • Поведенческие (Behavioral) - паттерны взаимодействия объектов (Chain of Responsibility, Command, Interpreter, Iterator, Mediator, Memento, Observer, State, Strategy, Template Method, Visitor)

⚙️ Код сервера (Google Apps Script)

Полный код вашего веб-приложения для развертывания в Google Apps Script:

function getDataSheet(table) {
  // Возвращает активный лист или лист по имени
  return SpreadsheetApp.getActiveSpreadsheet().getSheetByName(table);
}

function getAllRecords(table) {
  const sheet = getDataSheet(table);
  const allData = sheet.getDataRange().getValues();
  
  // Пропускаем заголовки
  const data = allData.slice(1);
  
  // Если есть заголовки, вернем как объекты
  if (allData.length > 0) {
    const headers = allData[0];
    return data.map(row => {
      const obj = {};
      headers.forEach((header, idx) => {
        obj[header] = row[idx];
      });
      return obj;
    });
  }
  
  return data;
}

// Функция для создания JSON ответа
function createJsonResponse(data) {
  return ContentService
    .createTextOutput(JSON.stringify(data))
    .setMimeType(ContentService.MimeType.JSON);
}

// Обработка GET запросов
function doGet(e) {
  // Добавляем проверку на существование параметров
  if (!e || !e.parameter) {
    // Если нет параметров, возвращаем приветственное сообщение или инструкцию
    return createJsonResponse({ 
      success: false, 
      error: 'Добро пожаловать в API Patterns!',
      instructions: {
        getAllPatterns: 'Используйте ?action=getAllPatterns для получения всех паттернов',
        getPatternById: 'Используйте ?action=getPatternById&id=1 для получения паттерна по ID',
        getCategories: 'Используйте ?action=getCategories для получения всех категорий',
        getPatternsByCategories: 'Используйте ?action=getPatternsByCategories&category_id=1 для получения паттернов по категории'
      }
    });
  }
  
  const action = e.parameter.action;
  
  try {
    if (action === 'getAllPatterns') {
      const patterns = getAllRecords('Patterns');
      const categories = getAllRecords('Categories');
      const categoryMap = {};
      categories.forEach(category => {
        categoryMap[category.id] = category.title;
      });
      const enrichedPatterns = patterns.map(pattern => {
        const enriched = { ...pattern };
        
        if (pattern.category_id && categoryMap[pattern.category_id]) {
          enriched.category_name = categoryMap[pattern.category_id];
        }
        
        return enriched;
      });
      
      return createJsonResponse({ 
        success: true, 
        data: enrichedPatterns,
      });
    }
    else if (action === 'getPatternById') {
      const id = e.parameter.id;
      if (!id) {
        return createJsonResponse({ 
          success: false, 
          error: 'Не указан параметр id. Используйте: ?action=getPatternById&id=1' 
        });
      }
      const data = getAllRecords('Patterns');
      const record = data.find(item => item.id == id);
      if (!record) {
        return createJsonResponse({ 
          success: false, 
          error: `Запись с ID ${id} не найдена` 
        });
      }
      return createJsonResponse({ success: true, data: record });
    }
    else if (action === 'getCategories'){
      const data = getAllRecords('Categories');
      return createJsonResponse({ success: true, data: data });
    }
    else if (action === 'getPatternsByCategories') {
      const category_id = e.parameter.category_id;
      if (!category_id) {
        return createJsonResponse({ 
          success: false, 
          error: 'Не указан параметр category_id. Используйте: ?action=getPatternsByCategories&category_id=1' 
        });
      }
      const patterns = getAllRecords('Patterns');
      const categories = getAllRecords('Categories');

      const category = categories.find(item => item.id == category_id);
      if (!category) {
        return createJsonResponse({ 
          success: false, 
          error: `Категория с ID ${category_id} не найдена` 
        });
      }
      const filtered_patterns = patterns.filter(pattern => pattern.category_id == category_id);
      return createJsonResponse({ success: true, data: filtered_patterns });
    }
    else {
      return createJsonResponse({ 
        success: false, 
        error: 'Неизвестное действие. Доступные действия: getAllPatterns, getPatternById, getCategories, getPatternsByCategories',
        available_actions: ['getAllPatterns', 'getPatternById', 'getCategories', 'getPatternsByCategories']
      });
    }
  } catch (error) {
    return createJsonResponse({ success: false, error: error.message });
  }
}

📝 Инструкция по развертыванию:

  1. Откройте Google Apps Script
  2. Создайте новый проект
  3. Вставьте код выше в редактор
  4. Создайте Google Sheets таблицу с двумя листами: "Patterns" и "Categories"
  5. Вставьте ID таблицы в код (если нужно) или убедитесь, что скрипт привязан к таблице
  6. Нажмите "Развернуть" → "Новое развертывание"
  7. Выберите "Веб-приложение" и установите доступ "Все, у кого есть ссылка"
  8. Скопируйте полученный URL для использования в API

⚠️ Коды ошибок

400 - Bad Request
Не указаны обязательные параметры или указан неверный action
404 - Not Found
Запись с указанным ID не найдена (паттерн или категория)
500 - Internal Server Error
Ошибка на стороне сервера (проблемы с доступом к таблице или парсингом данных)

💻 Примеры использования

JavaScript (Fetch API)

const baseUrl = 'https://script.google.com/macros/s/.../exec';

// Получение всех паттернов
fetch(`${baseUrl}?action=getAllPatterns`)
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      data.data.forEach(pattern => {
        console.log(`${pattern.title} (${pattern.term}): ${pattern.problem}`);
      });
    }
  });

// Получение паттерна по ID
const id = 1;
fetch(`${baseUrl}?action=getPatternById&id=${id}`)
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      console.log('Решение:', data.data.solution);
      console.log('Примеры:', data.data.examples);
    }
  });

// Получение всех категорий
fetch(`${baseUrl}?action=getCategories`)
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      console.log('Категории:', data.data);
    }
  });

// Получение паттернов по категории
const categoryId = 1;
fetch(`${baseUrl}?action=getPatternsByCategories&category_id=${categoryId}`)
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      console.log(`Паттерны категории ${categoryId}:`, data.data);
    }
  });

Python (Requests)

import requests

base_url = 'https://script.google.com/macros/s/.../exec'

# Получение всех паттернов
response = requests.get(base_url, params={'action': 'getAllPatterns'})
data = response.json()
if data['success']:
    for pattern in data['data']:
        print(f"{pattern['title']} - {pattern['term']}")
        print(f"  Проблема: {pattern['problem']}")
        print(f"  Решение: {pattern['solution']}")

# Получение паттерна по ID
response = requests.get(base_url, params={
    'action': 'getPatternById',
    'id': 1
})
data = response.json()
if data['success']:
    print(data['data']['solution'])

# Получение категорий
response = requests.get(base_url, params={'action': 'getCategories'})
data = response.json()
if data['success']:
    for category in data['data']:
        print(f"{category['id']}: {category['title']}")

# Получение паттернов по категории
response = requests.get(base_url, params={
    'action': 'getPatternsByCategories',
    'category_id': 1
})
data = response.json()
if data['success']:
    print(f"Найдено паттернов: {len(data['data'])}")
